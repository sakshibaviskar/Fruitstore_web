## 🌟 About The Project

**Sakshi’s Fruitstore** is a modern, stylish, and responsive fruit shop website designed to simulate a real-world e-commerce experience.  
It combines **clean UI**, **premium visuals**, and **smooth interactions** to create a realistic online shopping interface for fruits and dry fruits.

> 🍇 Designed to look like a real brand — not just a project.

---

## 🎯 Project Highlights

✨ Attractive premium homepage layout  
🍎 Real fruit & dry fruit product images  
🛒 Dynamic cart system  
❤️ Wishlist feature  
📦 Order summary page  
📱 Fully responsive design  
🎨 Elegant purple luxury theme  
⚡ Smooth animations & hover effects
